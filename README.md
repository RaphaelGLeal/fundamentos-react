# Projeto
-   Create
    - npx create-react-app %nome_projeto%
    - npm init react-app %nome_projeto%
    - yarn create react-app %nome_projeto%
-   Build
    - yarn build
-   Start
    - yarn start        
# Packages
-   yarn add @types/react
-   yarn add boostrap
-   yarn add @popperjs/core
